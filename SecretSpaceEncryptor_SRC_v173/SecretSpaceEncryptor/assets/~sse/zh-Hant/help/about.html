Secret Space Encryptor
（1.7.3）

  • 密碼管理員、文字加密和檔案加密等功能已整合為全方位解決方案。
  • 密碼保險箱：將所有密碼、PIN 和註釋資訊置於同一個安全位置用一個主密碼統一存儲和管理。提供匯入/匯出功能（使用經壓縮、完全加密的 .pwv 檔案格式，或未加密、可編輯的 .xml 檔案格式）。
  • 文字加密：為您的訊息、註釋和其他類型文字提供安全防護，避免意外被他人讀取。使用內部資料庫，或僅在本應用程式和您喜愛的其他應用程式之間相互複製/貼上。為當前加密/解密工作階段設定一個工作階段密碼，您也可以為任何目的（註釋、電子郵件、社交網路、與 A、B、C ……等人的通訊溝通記錄等）設定不限數量的密碼。
  • 檔案加密：安全加密您的私人/保密檔案或整個資料夾。
  • 工具箱：密碼生成器、清空剪貼簿、演算法效能評測……
  • 所有加密都採用強大加密演算法進行：目前支援 AES（Rijndael）256 位元、RC6 256 位元、Serpent 256 位元、Blowfish 448 位元、Twofish 256 位元、GOST 256 位元 + Threefish 1024 位元、SHACAL-2 512 位元（用於 S.S.E. Pro 版）等加密演算法。
  • 安裝執行本應用程式只需很小權限。沒有廣告。
  • 加密引擎基於 Bouncy Castle 加密庫（Java）和 Crypto++（C++）。
  • 應用程式首頁（見下載一節）上提供了文字加密（Text Encryptor）和檔案加密（File Encryptor）的跨平台桌面版本（Windows、Linux、Mac OS X……）。
  • 在 P.T.E. for iOS 提供了 Paranoia Text Encryption for iOS（支援 iPhone/iPad/iPod）。
  • 在線（基於網絡）文本加密工具：https://pteo.paranoiaworks.mobi/cn
  • 本軟體為開源軟體。
  • (EN) Detailed format specifications: Password Vault, Text Encryptor, File Encryptor
