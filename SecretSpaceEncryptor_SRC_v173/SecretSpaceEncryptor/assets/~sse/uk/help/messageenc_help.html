Шифрувач тексту
(натисніть літеру або перегорніть вниз для перегляду розділів)

Інші розділи

  • Швидкий початок
  • Пароль сесії
  • Забутий пароль
  • Системний буфер обміну
    •  Інтернет версія (AES)

Швидкий початок

    Шифрування
    • Задайте пароль для поточної сесії.
    • Наберіть або вставте текст в поле "Розшифрований текст" (секція C на малюнку).
    • Натисніть кнопку Шифрувати.
    • Зашифрований текст з'явиться в полі "Зашифрований текст" (секція E на малюнку).
    • Скопіюйте зашифрований текст в системний буфер обміну та використовуйте його де забажаєте або збережіть його в базі даних додатку для пізнішого використання (кнопка F).

    Розшифрування
    • Задайте пароль для поточної сесії.
    • Наберіть або вставте зашифрований текст в полі "Зашифрованный текст" (секція E на малюнку).
    • Натисніть кнопку Розшифрувати. (примітка: алгоритм розшифрування визначається автоматично)
    • Расшифрованный текст появится в поле "Расшифрованный текст" (секция C на малюнку).
    • Скопіюйте розшифрований текст в системний буфер обміну та використовуйте його де забажаєте.

Пароль сесії

  • Додаток Шифрувач тексту непрацює як стандартний "Додаток сейф". Пароль не встановлюється на вхід в додаток, але задається для поточної сесії шифрування/розшифрування. Ви можете мати необмежену кількість паролів для будь-яких цілей (заміток, пошти, спілкування з людьми A B C, ...).

Забутий пароль

  • Не існує "чорного входу" для розшифрування будь-якого тексту без правильного пароля. Якщо ви забули ваш пароль... звиняйте ;-)

Системний буфер обміну

  • Дейкі системи використовують назву область монтажу (pasteboard).
  • Скористайтесь пошуком Google для додаткової інформації - введіть щось на зразок: android using copy paste

A - Копіювати/Вставити текст

  • копіює текст з поля "Незашифрований текст" в системний буфер обміну
  • вставляє текст з системного буферу обміну в поле "Незашифрований текст"

B - Шифрувати/Розшифрувати текст

  • шифрує текст в поле "Незашифрований текст" та поміщає результат в полі "Зашифрований текст"
  • розшифровує зашифровує текст в полі "Зашифрований текст" та поміщає результат в полі "Ррзшифрований текст"

C - Поле розшифрованого тексту

  • помістіть сюди текст, який бажаєте розшифрувати

D - Кнопка Задати пароль

  • Задає Пароль сесії.
  • колір цієї кнопки залежить ві поточної стійкості пароля (червоний - слабкий, жовтий - добрий, зелений - сильний)

E - Поле зашифрованого тексту

  • помістіть сюди текст, який бажаєте розшифрувати

F - Кнопка Ще

  • Завантажити текст з БД або файлу: завантажує попередньо збережені дані з бази даних додатка (або з файлу)
  • Зберегти текст в БД або файл: зберігає дані в полі "Зашифрований текст" в базі даних додатка (або в файл) для використання пізніше
  • Видалити текст з БД або видалити файл: видаляє обрані дані з бази даних додатка (або видаляє файл)
  • + Стеганографія (приховування тексту в зображенні - JPG, алгоритм F5)

G - Копіювати/Вставити зашифрований текст

  • копіює текст з поля "Зашифрований текст" в системний буфер обміну
  • вставляє текст з системного буферу обміну в поле "Зашифрований текст"

H - Стерти текстові поля

  • Очищає текстові поля.

I - Відправити текст в додаток

  • Відправляє зашифрований текст в обраний додаток.
