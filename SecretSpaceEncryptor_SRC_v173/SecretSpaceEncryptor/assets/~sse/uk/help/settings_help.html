A - Доступні налаштування

  • Сейф паролів
    • Алгоритм: алгоритм шифрування для Сейфу паролів
    • Розмір шрифта в списку паролів: розмір тексту в списку (розмір імені пароля та пароля)
    • Папка авторезервування: Якщо увімкнено, дані сейфу паролів буде експортовано (файл PWV_AutoBackup.pwv) до обраної папки при кожній зміні вмісту сейфу паролів. Наприклад, він може збережений в хмарному сховищі.
    • Заблокувати екран: блокує экран Сейфу паролів коли додаток переходить в фон (коли додаток "згорнуто")
  • Шифрувач тексту
    • Алгоритм: алгоритм шифрування для Шифрувача тексту
    • Розмір шрифта: Розмір текста в полях незашифрованого/зашифрованого текста.
    • Сховати інформаційні повідомлення: ховає найменш важливі спливаючі повідомлення (такі як: текст було скопійовано в...)
    • Кінцеве кодування: Кодування вихідного тексту. Доступне кодування: Base64 (за замовчуванням), Base32 (RFC 4648), Тільки цифри (0-9).
    • Додати пробіл: Додати пробіл після кожного N-го символу (до кодованих текстів).
    • Заблокувати екран: блокує экран Шифрувача тексту коли додаток переходить в фон (коли додаток "згорнуто")
  • Шифрувач файлів
    • Алгоритм: алгоритм шифрування для Шифрувача файлів
    • Множинний вибір файлів (тільки для версії PRO): Активує/Деактивує режим множинного вибору файлів (пакетна обробка)
    • Кінцева папка для зашифрованих файлів:
      Така ж як джерело: Вихідні файли поміщаються в ту ж папку де розташовані вхідні.
      Поперед визначене розташування: Використовуйте кнопку "Налаштовувані папки" (у верхньому лівому куті - ) в Шифрувачі файлів для задання розташування (Задати поточну папку як... → Папка зашифрованих файлів).
      Спитати: Додаток спитає вас про розташування вихідних файлів перед кожним процесом шифрування.
    • Кінцева папка для розшифрованих файлів: Таке ж виконання як і "Кінцева папка для зашифрованих файлів".
    • Зачищати джерело після шифрування: Видаляє (із зачищенням) вихідні файли автоматично після шифрування
    • Видаляти файл .enc після розшифрування: Видаляти вихідний файл (зашифрований файл .enc) автоматично після розшифрування.
    • Показати попередження про заміну файла(-ів) .enc: Показати діалог "продовжити/скасувати" коли файл(-и) .enc із завданим іменем вже існує(-ють).
    • Розмір ескізів: Встановлює розмір ескізів зображень або вимикає ескізи.
    • Заблокувати екран: заблокувати екран Шифрувача файлів коли додаток переходить в фон (коли додаток "згорнуто") - Пароль сесії має бути задано
  • Загальні налаштування
    • Шлях імпорту/експорту: папка додатка для імпорту/експорту
    • Захист запуску додатка: Запитувати аутентифікaцію для кожного запуску додатка. Цей пароль використовується тільки для аутентифікaції запуску. Він не впливає на функції шифрування, а також на функції, що відносяться до блокування екрану. Тільки похідна (вихід складних функцій KDF) пароля аутентифікaції буде збережена, проте не рекомендується використовувати однаковий пароль для цілей шифрування.
    • Перешкоджати створенню знімків екрану: Розглядати вміст вікна як надійне, запобігаючи його від появи на знімках екрану. (потребує перезапуску)
    • Показувати діалог виходу з модулів додатку: підтверджувати повернення в головне меню (тільки при натисканні кнопки Назад, пряме натискання кнопки "Головне меню" не впливає)
    • Дозволити паролі Юнікод: Тільки символи ASCII 32-126 дозволені за замовченням ( => сумісність - практично кожен пристрій/система має достатню клавіатуру). Якщо ви дозволяєте "Паролі Юнікод", ви можете використовувати кожен символ, який дозволяє ваша клавіатура.
    • Вимкнути алгоритми на власному коді: використовувати тільки незалежну від платформи версію (тільки власний код може викликати проблеми з цим пристроєм)
