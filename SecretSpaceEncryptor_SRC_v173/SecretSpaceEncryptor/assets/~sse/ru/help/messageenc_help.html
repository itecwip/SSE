Быстрый старт

    Шифрование
    • Задайте пароль для текущей сессии.
    • Наберите или вставте текст в поле "Расшифрованный текст" (секция C на рисунке).
    • Нажмите кнопку Шифровать.
    • Зашифрованный текст появится в поле "Зашифрованный текст" (секция E на рисунке).
    • Скопируйте зашифрованный текст в системный буфер обмена и используйте его где хотите или сохраните его в базе данных приложения для использования позднее (кнопка F).

    Расшифровка
    • Задайте пароль для текущей сессии.
    • Наберите или вставьте зашифрованный текст в поле "Зашифрованный текст" (секция E на рисунке).
    • Нажмите кнопка Расшифровать. (примечание: алгоритм расшифровки определяется автоматически)
    • Расшифрованный текст появится в поле "Расшифрованный текст" (секция C на рисунке).
    • Скопируйте расшифрованный текст в системный буфер обмена и используйте его где захотите.

Пароль сессии

  • Приложение Шифратор текста не работает как стандартное "Приложение сейф". Пароль не устанавливается на вход в приложение, но задается для текущей сессии шифрования/расшифровки. Вы можете иметь неограниченное количество паролей для любых целей (заметок, почты, общения с личностями A B C, ...).

Забытый пароль

  • Не существует "черного хода" для расшифровки любого текста без правильного пароля. Если вы забыли ваш пароль... простите;-)

Системный буфер обмена

  • Некоторые системы используют название область монтажа (pasteboard).
  • Воспользуйтесь поиском Google для дополнительной информации - введите что-то типа: android using copy paste

A - Копировать/Вставить текст

  • копирует текст из поля "Незашифрованный текст" в системный буфер обмена
  • вставляет текст из системного буфера обмена в поле "Незашифрованный текст"

B - Шифровать/Расшифровать текст

  • шифрует текст в поле "Незашифрованный текст" и поместить результат в поле "Зашифрованный текст"
  • расшифровывает зашифрованный текст в поле "Зашифрованный текст" и поместить результат в поле "Расшифрованный текст"

C - Поле расшифрованного текста

  • поместите сюда текст, который хотите зашифровать

D - Кнопка Задать пароль

  • Задает Пароль сессии.
  • цвет этой кнопки зависит от текущей стойкости пароля (красный - слабый, желтый - хороший, green - сильный)

E - Поле зашифрованного текста

  • поместите сюда текст, который хотите расшифровать

F - Кнопка Еще

  • Загрузить текст из БД или файла: загружает предварительно сохраненные данные из базы данных приложения (или из файла)
  • Сохранить текст в БД или файл: сохраняет данные в поле "Зашифрованный текст" в базу данных приложения (или в файл) для использования позднее
  • Удалить текст из БД или удалить файл: удаляет выбранные данные из базы данных приложения (или удаляет файл)
  • + Стеганография (скрытие текста в изображении - JPG, алгоритм F5)

G - Копировать/Вставить зашифрованный текст

  • копирует текст из поля "Зашифрованный текст" в системный буфер обмена
  • вставляет текст из системного буфера обмена в поле "Зашифрованный текст"

H - Стереть текстовые поля

  • Очищает текстовые поля.

I - Отправить текст в приложение

  • Отправляет зашифрованный текст в выбранное приложение.
