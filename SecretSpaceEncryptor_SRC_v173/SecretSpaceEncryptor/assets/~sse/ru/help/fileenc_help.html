Быстрый старт

Шифрование
  • Выберите файл или папку (нажмите файл, долгое нажатие для папки).
  • Нажмите кнопку Шифровать (нижний правый угол).
  • Задайте пароль.
  • Новый файл ".enc" будет создан.
    • (Примечание: Вы можете удалить/зачистить оригинальный файл(ы) после шифрования или это можно сделать автоматически: Настройки: Шифратор файлов → Зачищать источник после шифрования)

Расшифровка
  • Выберите зашифрованный файл (нажмите файл).
  • Нажмите кнопку Расшифровать (нижний правый угол).
  • Задайте пароль.
  • Файл или папка будут расшифрованы.

A - Доступные диски

  • ярлыки на доступные диски на текущем устройстве + определенные пользователем папки Домашняя/По умолчанию
  • Настраиваемые папки -
    • Папки вывода зашифрованных/расшифрованных файлов можно задать здесь (Если включено Настройки: Шифрователь файлов → Конечная папка для зашифрованных/расшифрованных файлов)

  • Кнопка SAF (Android 5+)

B - Параметры текущего тома

  • доступное место / размер тома

C - Выбранный файл/папка

  • долгий клик для выбора папки
  • короткий клик для выбора файла
  • Выбранный файл или папка может быть:
    • Зашифрован или Расшифрован (в соответствии со статусом выбранного файла) с помощью кнопки Шифровать/Расшифровать (E)
    • Переименован, Открыт, Удален или Зачищен с использование кнопки "Меню" на вашем устройстве
    • Примечание: долгое нажатие на файле приводит к такому же поведению к и использование кнопки "Меню" на выбранном файле/папке

D - Зашифрованный файл

  • зашифрованный файл может быть распознан по: расширению .enc, окрашенному фиолетовым фоном и значке "замок"

E - Кнопка Шифровать/Расшифровать

  • шифрует или расшифровывает (в соответствии с текущим выбранным элементом) файл или папку

F - Сжатие ВКЛЮЧЕНО/ВЫКЛЮЧЕНО

  • активирует/деактивирует сжатие данных

G - Направляет в Главное меню

  • переходит прямо в Главное меню приложения

H - Кнопка Еще

  • Выбранный файл/папка может быть Переименован, Удален, Зачищен или Открыт (файл)

I - Задать пароль сессии

  • Установить пароль ("мастер") сессии.
  • Пароль не устанавливается для входа в приложение, но устанавливается на текущую сессию шифрования/расшифровки. Вы можете иметь неограниченное количество паролей для любых целей.

    Режим SAF (Android 5+)

    Предисловие
    В Android 4.4 (KitKat) - Google внезапно отключил все сторонние (не предустановленные) приложения от разрешения на запись на внешнее/снимаемое хранилище. Благодаря этому решению много приложений были ущемлены и некоторые из них полностью стали бесполезны. Давление на Google привело к копромисному решению, внедренному в Android 5. Приложение может восстановить разрешение на запись на внешнее хранилище, используя новый API, вместе с SAF (Storage Access Framework).

    Первый запуск
    • Будет показан выбор папок. Вам нужно выбрать папку (или том) с которой вы хотите работать.
    • Нажмите кнопку Еще (верхний правый угол) и выберите Показать SD-карту.
    • Нажмите кнопку Меню (верхний левый угол) и выберите хранилище, которое хотите использовать.
    • Нажмите ВЫБРАТЬ (внизу).
    Ограничения
    Есть некоторые ограничения режима SAF, о которых вы должны знать.
    • Домашняя папка не может быть задана в SAF, при этом последний выбранный SAF URI автоматически сохраняется: Кнопка Домой → Перейти к последней выбранной папке SAF.
    • Функция Зачистка не работает - Если опция "Зачищать после шифрования" включена в настройках приложения, файл(ы) будет только удален.
    • Режим SAF обеспечивает низкую производительность по сравнению со стандартным режимом прямого файлового доступа (особенно с папками, содержащими много файлов/подпапок).
    • Функция открыть файл будет работать только если конечное приложение будет способно распознать SAF URI.

    Часто задаваемые вопросы по Шифратору файлов

    Проблема: KitKat (Android 4.4) - Внешняя SD-карта только для чтения - сообщение: EACCESS (Разрешение отклонено).

    Ответ: Android 4.4 не разрешает доступ записи на внешнюю SD-карту какому-либо приложению, за исключение одному из предустановленных (предустановленных производителем устройства (или системным образом)). Мы также абсолютно ничего не можем сделать с этой проблемой. Единственное решение - это получить права администратора вашего устройства и изменить разрешения или обновиться до Android 5 и воспользоваться SAF.

    Воспользуйтесь поисковиком Google для дополнительной информации - введите что-то типа: android 4.4 external card read only

    Альтернативно - Создайте следующую папку (на вашей внешней SD-карте):
    Android/data/
    com.paranoiaworks.unicus.android.sse

    Приложение должно получить разрешения записи в эту папку.
