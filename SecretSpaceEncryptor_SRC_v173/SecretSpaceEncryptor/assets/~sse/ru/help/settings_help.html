A - Доступные настройки

  • Сейф паролей
    • Алгоритм: алгоритм шифрования для Сейфа паролей
    • Размер шрифта в списке паролей: размер текста в списке (размер имени пароля и пароля)
    • Папка авторезервирования: Если включено, данные сейфа паролей будут экспортированы (файл PWV_AutoBackup.pwv) в выбранную папку при каждом изменении содержимого сейфа паролей. Например, он может быть сохранен в облачном хранилище.
    • Заблокировать экран: блокирует экран Сейфа паролей когда приложение переходит в фон (когда приложение "свернуто")
  • Шифратор текста
    • Алгоритм: алгоритм шифрования для Шифратора текста
    • Размер шрифта: Размер текста в полях незашифрованного/зашифрованного текста.
    • Скрывать информационные сообщения: скрывает наименее значимые всплывающие сообщения (такие как: текст был скопирован в...)
    • Конечная кодировка: Кодировка выходного текста. Доступные кодировки: Base64 (по умолчанию), Base32 (RFC 4648), Только цифры (0-9).
    • Добавить пробел: Добавить пробел после каждого N-го символа (к кодируемым текстам).
    • Заблокировать экран: блокирует экран Шифраторa текста когда приложение переходит в фон (когда приложение "свернуто")
  • Шифратор файлов
    • Алгоритм: алгоритм шифрования для Шифратора файлов
    • Множественный выбор файлов (только для версии PRO): Активировать/Деактивировать режим множественного выбора файлов (пакетной обработки)
    • Конечная папка для зашифрованных файлов:
      Такая же как исходная: Выходные файлы помещаются в ту же папку где расположены входные.
      Предопределенное расположение: Используйте кнопку "Настраиваемые папки" (в верхнем левом углу - ) в Шифраторе файлов для задания расположения (Задать текущую папку как... → Папка зашифрованных файлов).
      Спросить: Приложение спросит вас о расположении выходных файлов перед каждым процессом шифрования.
    • Конечная папка для расшифрованных файлов: Такое же исполнение как и "Конечная папка для зашифрованных файлов".
    • Зачищать источник после шифрования: Удаляет (с зачисткой) исходные файлы автоматически после шифрования.
    • Удалять файл .enc после расшифровки: Удалять исходный файл (зашифрованный файл .enc) автоматически после расшифровки.
    • Показать предупреждение о замене файла(-ов) .enc: Показать диалог "продолжить/отмена" когда файл(-ы) .enc с заданным именем уже существует(-ют).
    • Размер эскизов: Устанавливает размер изображений эскизов или выключает эскизы.
    • Заблокировать экран: заблокировать экран Шифратора файлов когда приложение переходит в фон (когда приложение "свернуто") - Пароль сессии должен быть задан
  • Общие настройки
    • Путь импорта/экспорта: папка приложения для импорта/экспорта
    • Защита запуска приложения: Запрашивать аутентификaцию для каждого запуска приложения. Этот пароль используется только для аутентификaции запуска. Он не влияет на функции шифрования, а также на функции, относящиеся к блокировке экрана. Только производная (выход сложных функций KDF) пароля аутентификaции будет сохранена, однако не рекомендуется использовать одинаковый пароль для целей шифрования.
    • Препятствовать созданию снимков экрана: Рассматривать содержимое окна как надежное, предотвращая его от появления на снимках экрана. (требуется перезапуск)
    • Показывать диалог выхода из модулей приложения: подтверждать возврат в главное меню (только при нажатии кнопки Назад, прямой нажатие кнопок "Главное меню" не влияет)
    • Разрешить пароли Юникод: Только символы ASCII 32-126 разрешены по умолчанию ( => совместимость - практически каждое устройство/система имеет достаточную клавиатуру). Если вы разрешаете "Пароли Юникод", вы можете использовать каждый символ, который позволяет ваша клавиатура.
    • Выключить алгоритмы на собственном коде: использовать только не зависимую от платформы версию (только собственный код может вызывать проблемы с этим устройством)
