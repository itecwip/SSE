A - Password Folders

  • Short Click on Folder: Open your password item list:
     
     
    • Short Click on Item: Show/Hide password
    • Long Click on Item: Show/Edit parameters (title, password, notes, and color); Delete/Move To item
    • Icon meanings:
      - Copy to Clipboard
      - Copy to Clipboard (All/Chosen Paragraph—paragraphs end with the Enter key)
      - Password Generator
  • Long Click on Folder: Show/Edit/Delete folder parameters (name, notes, and color);
     
    • Enforce Position (Top-Right Corner): (ABC...) This means alphabetical order (default); numbers refer to the absolute position of the edited folder. (Note: If you choose a position that already exists, the original folder will be replaced in its position by the edited one, and the original folder position will be automatically set to default (alphabetical) mode.)

B - More Options Button

    Note: Press the “Menu button” or “Expand button” (bottom-right corner) to show the Bottom Menu.
  • Change Password: Change the “master” password for Password Vault.
  • Import Vault: Import the previously exported Password Vault data from the application Import/Export Directory.
  • Merge Vaults: Merge current data and data from a selected file.
  • Export Vault: Export (backup) the current Password Vault data to the application Import/Export Directory.
  • Notes:
    • Application Import/Export Directory can be defined in the Application's Settings
    • You can choose between a compressed, fully-encrypted .pwv file format and an unencrypted, editable .xml format.

C - New Folder

    Note: Press the “Menu button” or “Expand button” (bottom-right corner) to show the Bottom Menu.
  • Create a new password folder using the same dialog as the Show/Edit/Delete feature.

More...

  • Whole vault is encrypted as one object. All attributes (name, password, notes, etc.) are stored as secure as the password itself.
  • Forgotten Password
    • There is no way to enter your Password Vault in case you forget your password.
    • You can only click on the Cancel button in the login (password) dialog and follow the “Help me get in!” feature.
