Quick Start

    Encryption
    • Set a password for the current session.
    • Write or paste a text into the "Unencrypted Text" area (Section C within the image).
    • Press the Encrypt button.
    • The encrypted text will appear in the "Encrypted Text" area (Section E within the image).
    • Copy the encrypted text to the system clipboard and use it wherever you want or save it to the app database for later use (F button).

    Decryption
    • Set a password for the current session.
    • Write or paste an encrypted text into the "Encrypted Text" area (Section E within the image).
    • Press the Decrypt Button. (Note: Decryption algorithm is detected automatically.)
    • The decrypted text will appear in the "Unencrypted Text" area (Section C within the image).
    • Copy the unencrypted text to the system clipboard and use it wherever you want.

Steganography

  • A steganographic feature is included (concealing a text within an image - JPG). A steganographic algorithm (F5 algorithm) is used in combination with chosen symmetric cipher algorithm to make a final steganogram (JPEG image).
  • note: Some applications for communication (like WhatsApp, Viber, Messenger, ...) can resize or recompress images to lower bandwidth. Concealed data are corrupted during the process => do not use these applications to send steganograms.

    Make Steganogram
    • Set a password for the current session.
    • Write or paste a text you want to encrypt (and conceal) into the "Unencrypted Text" area (Section C within the image).
    • Tap the "More" button (bottom-right corner - F).
    • Choose "Encrypt and Make Steganogram".
    • Select a carrier image (jpg, png, bmp).
    • Tap the "Continue" button.
    • The steganogram will be made and saved (please be patient, it can take some time).

    Decrypt from Steganogram
    • Set a password for the current session.
    • Tap the "More" button (bottom-right corner - F).
    • Choose "Decrypt from Steganogram".
    • Select a steganogram.
    • The decrypted text will appear in the "Unencrypted Text" area (Section C within the image).
    • Copy the unencrypted text to the system clipboard and use it wherever you want.

Session Password

  • The Text Encryptor app does not work as a standard "Vault application". A password is not set to enter the application, but set for the current encryption/decryption session. You may have an unlimited number of passwords for any purpose (notes, emails, communication with persons A, B, C, etc.).

Forgotten Password

  • There is no “backdoor” way to decrypt any text without a correct password. If you forget your password, “sorry.” ;-)

System Clipboard

  • Some other systems use the name "pasteboard".
  • Just use Google for more information, for example, search for something like: android using copy paste

A - Copy/Paste Text

  • Copy text from the "Unencrypted Text" area to the system clipboard.
  • Paste text from the system clipboard to the "Unencrypted Text" area.

B - Encrypt/Decrypt Text

  • Encrypt text in the "Unencrypted Text" area and put the result into the "Encrypted Text" area.
  • Decrypt encrypted text in the "Encrypted Text" area and put the result into the "Unencrypted Text" area.

C - Unencrypted Text Area

  • Place the text you want to encrypt here.

D - Set Password Button

  • Set a session password.
  • The color of this button depends on the current password strength (red = weak, yellow = fair, and green = strong).

E - Encrypted Text Area

  • Place the text you want to decrypt here.
  • The text in this area is red when one of the text fields (unenc/enc) is changed, so the data in the other field does not correspond to the actual value.
  • The text in this area is blue when the encrypted text "has not been used" (copied, saved).

F - More Options Button

  • Load Text from DB or from File: Load previously saved data from the application database (or from a file).
  • Save Text to DB or to File: Save data in the "Encrypted Text" area to the application database (or to a file) for later use.
  • Delete Text from DB or Delete File: Delete selected data from the application database (or delete a file).
  • + steganographic features

G - Copy/Paste Encrypted Text

  • Copy text from the "Encrypted Text" area to the system clipboard.
  • Paste text from the system clipboard to the "Encrypted Text" area.

H - Erase Text Areas

  • Clear text areas.

I - Send Text to Application

  • Send encrypted text to the chosen application.

Text Encryption FAQ

Issue: The message is decrypted even when I change the algorithm in the app settings. How is it possible?

Answer: You can set an ENCRYPTION algorithm (the decryption algorithm is detected automatically - the information about the used algorithm is part of the app data format).


Issue: The application produces a completely different output (using the same input) whenever I tap the encrypt button. How is it possible?

Answer: Two words for Google: cryptographic salt
