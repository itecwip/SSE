秘密空间加密器
(Secret Space Encryptor)
（1.7.3）

  • 密码管理器、文本加密和文件加密等功能已集成为一体化解决方案。
  • 密码保险箱：将所有密码、PIN 和注释信息置于同一个安全位置用一个主密码统一存储和管理。提供导入/导出功能（使用经压缩、完全加密的 .pwv 文件格式，或未加密、可编辑的 .xml 文件格式）。
  • 文本加密：为您的消息、注释和其他类型文本提供安全防护，避免意外被他人读取。使用内部数据库，或仅在本应用和您喜爱的其他应用之间相互复制/粘贴。为当前加密/解密会话设置一个会话密码，您也可以为任何目的（注释、电子邮件、社交网络、与 A、B、C ……等人的通讯沟通记录等）设置不限数量的密码。
  • 文件加密：安全加密您的私人/保密文件或整个文件夹。
  • 工具箱：密码生成器、清空剪贴板、算法性能评测……
  • 所有加密都采用强大加密算法进行：目前支持 AES（Rijndael）256 位、RC6 256 位、Serpent 256 位、Blowfish 448 位、Twofish 256 位、GOST 256 位 + Threefish 1024 位、SHACAL-2 512 位（用于 S.S.E. Pro 版）等加密算法。
  • 安装运行本应用只需很小权限。没有广告。
  • 加密引擎基于 Bouncy Castle 加密库（Java）和 Crypto++（C++）。
  • 应用主页（见下载一节）上提供了文本加密（Text Encryptor）和文件加密（File Encryptor）的跨平台桌面版本（Windows、Linux、Mac OS X……）。
  • 在 P.T.E. for iOS 提供了 Paranoia Text Encryption for iOS（支持 iPhone/iPad/iPod）。
  • 在线（基于网络）文本加密工具：https://pteo.paranoiaworks.mobi/cn
  • 本软件为开源软件。
  • (EN) Detailed format specifications: Password Vault, Text Encryptor, File Encryptor
