快速入门

    加密
    • 为当前会话设置一个密码。
    • 在“未加密文本”区域（上图中的 C 区）写入或贴入文本。
    • 按下“加密”按钮。
    • “加密文本”区域（上图中的 E 区）中将出现经加密的文本。
    • 将加密文本复制到系统剪贴板中，可随意使用，也可以将其保存到应用数据库中供以后使用（F 按钮）。

    解密
    • 为当前会话设置一个密码。
    • 在“加密文本”区域（上图中的 E 区）写入或贴入加密文本。
    • 按下“解密”按钮。（注：解密算法被自动检测）
    • “未加密文本”区域（上图中的 C 区）中将出现解密后的文本。
    • 将已解密的文本复制到系统剪贴板中，可随意使用。

会话密码

  • “文本加密”应用与“密码保险箱”应用的工作方式不同。“文本加密”应用未设置进入应用的密码，但设置了当前加密/解密会话的密码。您可以为任何目的（注释、电子邮件、社交网络、与 A、B、C...等人的通讯沟通记录等....）设置不限数量的密码。

忘记密码

  • 如没有正确的密码，不能通过任何“后门”来解密任何文本。如果您忘记了密码，“抱歉。” ;-)

系统剪贴板

  • 某些其他系统使用的名称是“粘贴板”。
  • 如需了解更多信息，建议使用谷歌搜索查找此类信息：android using copy paste（android 使用 复制 粘贴）

A - 复制/粘贴文本

  • 复制：将“未加密文本”区内的文本复制到系统剪贴板上。
  • 粘贴：将系统剪贴板上的文本粘贴到“未加密文本”区中。

B - 加密/解密文本

  • 加密：加密“未加密文本”区内的文本，并将结果置于“加密文本”区中。
  • 解密：解密“加密文本”区内的文本，并将结果置于“未加密文本”区中。

C - 未加密文本区

  • 在此放置需要加密的文本。

D - 设置密码按钮

  • 设置会话密码.
  • 此按钮的颜色取决于当前密码的强度（红色=弱，黄色=不错，绿色=强）。

E - 加密文本区

  • 在此放置需要解密的文本。
  • 当其中一个文本字段（未加密/加密）改变时，此区域的文本为红色，所以其它字段的数据不对应于实际值。
  • 当加密文本未被使用（复制、保存）时，此区域的文字为蓝色。

F - 更多选项按钮

  • 从数据库或文件中加载文本：从应用数据库（或一个文件）中加载以前保存的数据。
  • 将文本保存到数据库或文件：将“加密文本”区内的数据保存到应用数据库（或一个文件）中，以便以后使用。
  • 删除数据库中的文本或删除文件：删除应用数据库中的选定数据（或删除一个文件）。
  • + 隐写术 (steganography)（在图像中隐藏文本 - JPG, F5 算法）。

G - 复制/粘贴加密文本

  • 复制：将“加密文本”区内的文本复制到系统剪贴板上。
  • 粘贴：将系统剪贴板上的文本粘贴到“加密文本”区中。

H - 擦除文本区

  • 清空文本区。

I - 发送文本到应用

  • 将加密文本发送到选定的应用。
