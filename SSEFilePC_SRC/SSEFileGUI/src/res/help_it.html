UTILIZZO:
1. Inserire la password.
2. Scegliere un algoritmo di cifratura.
3. Trascinare i file o le cartelle e rilasciarli nell'area con riquadro rosso.
4. I file o le cartelle non crittografati verranno crittografati. I file crittografati verranno decrittografati.
